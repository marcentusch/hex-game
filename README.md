## Notes

[Hexagon Bible](https://www.redblobgames.com/grids/hexagons/)

- hexSize: distance from center to any corner
- hex height: `h = sqrt(3) * hexSize`
- hex width: `w = hexSize * 2`

## To run game

`love ./src`

## Colors

- black: 34,35,35
- white: 240, 246, 240
